```
api/src
├─ main
│   ├─ main
│   │   ├─ java.kyh.api.controller
│   │   │   └─ controller...
│   │   ├─ java.kyh.api.service
│   │   │   └─ service...
│   │   ├─ java.kyh.api.repository
│   │   │   └─ repository...
│   │   ├─ java.kyh.api.domain.entity
│   │   │   └─ entity...
│   │   ├─ java.kyh.api.domain.dto
│   │   │   └─ dto...
│   │   ├─ java.kyh.api.domain.type
│   │   │   └─ type...
│   │   └─ java.kyh.api.config
│   │       └─ config...
│   └─ resources
│       └─ application.properties
└─ test
    └─ 테스트 코드...
```
